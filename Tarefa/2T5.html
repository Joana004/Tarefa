Introdução ao CSS

INTRODUÇÃO AO CSS

CSS (Cascading Style Sheets) é uma linguagem utilizada para definir a aparência visual das páginas web.

IMPORTÂNCIA DO CSS

Permite separar o conteúdo do design, tornando os sites mais organizados e fáceis de manter.

TIPOS DE CSS

  • Inline — aplicado diretamente na tag HTML.
  • Interno — definido dentro da página.
  • Externo — arquivo .css separado.

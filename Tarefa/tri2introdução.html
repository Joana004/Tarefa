INTRODUÇÃO AO CSS

CSS (Cascading Style Sheets — Folhas de Estilo em Cascata) é uma linguagem utilizada para definir a apresentação visual de páginas web desenvolvidas com HTML. Enquanto o HTML é responsável pela estrutura do conteúdo (títulos, parágrafos, imagens e tabelas), o CSS controla a aparência desses elementos, como cores, fontes, tamanhos, espaçamentos e posicionamento.

IMPORTÂNCIA DO CSS NO DESENVOLVIMENTO WEB

O CSS é fundamental no desenvolvimento web moderno porque permite a separação entre conteúdo e design. Isso torna os sites mais organizados, fáceis de manter e visualmente atrativos.

  • Melhora a aparência visual dos sites
  • Facilita a manutenção e atualização do layout
  • Permite reutilização de estilos em várias páginas
  • Torna os sites mais responsivos e adaptáveis a diferentes dispositivos
  • Reduz a repetição de código HTML

TIPOS DE FOLHAS DE ESTILO (CSS)

  • CSS Inline: Aplicado diretamente dentro da tag HTML usando o atributo style. É indicado apenas para pequenos ajustes.
  • CSS Interno (ou Incorporado): Definido dentro da própria página HTML na seção head usando a tag style.
  • CSS Externo: Criado em um arquivo separado com extensão .css e ligado ao HTML. É o método mais recomendado.

Introdução ao CSS

INTRODUÇÃO AO CSS

CSS (Cascading Style Sheets — Folhas de Estilo em Cascata) é uma linguagem utilizada para definir a apresentação visual de páginas web desenvolvidas com HTML. Enquanto o HTML é responsável pela estrutura do conteúdo (títulos, parágrafos, imagens, tabelas), o CSS controla a aparência desses elementos, como cores, fontes, tamanhos, espaçamentos e posicionamento.

IMPORTÂNCIA DO CSS NO DESENVOLVIMENTO WEB

O CSS é fundamental no desenvolvimento web moderno porque permite a separação entre conteúdo e design. Isso torna os sites mais organizados, fáceis de manter e visualmente atrativos. Entre as principais importâncias do CSS destacam-se: - Melhora a aparência visual dos sites; Facilita a manutenção e atualização do layout; - Permite reutilização de estilos em várias páginas; Torna os sites mais responsivos e adaptáveis a diferentes dispositivos; - Reduz a repetição de código HTML.

TIPOS DE FOLHAS DE ESTILO (CSS)

Existem três tipos principais de folhas de estilo CSS:

  • CSS Inline - O CSS inline é aplicado diretamente dentro da tag HTML, usando o atributo style. É indicado apenas para pequenos ajustes, pois dificulta a manutenção do código.
  • CSS Interno (ou Incorporado) - O CSS interno é definido dentro da própria página HTML, na seção "head", usando a tag "style". É útil quando os estilos são aplicados apenas a uma única página.
  • CSS Externo - O CSS externo é criado em um arquivo separado com extensão .css e ligado ao HTML. É o método mais recomendado, pois facilita a organização, manutenção e reutilização dos estilos em várias páginas.

Continuação(Sintaxe Bássica do Css)

SINTAXE BÁSICA do CSS

A estrutura básica de uma regra CSS é composta por seletor, propriedade e valor. Exemplo: p { color: blue; font-size: 16px; }

TIPOS DE SELECTORES SIMPLES

  • Seletor de Elemento: Seleciona todas as tags HTML do mesmo tipo. Exemplo: p { color: red; }
  • Seletor de Classe: Usa o ponto (.) e pode ser reutilizado em vários elementos. Exemplo: .titulo { color: green; }
  • Seletor de ID: Usa o símbolo # e identifica um único elemento. Exemplo: #menu { background: black; }
  • Seletor Universal: Usa * para selecionar todos os elementos. Exemplo: * { margin: 0; padding: 0; }

Diferença entre Class e ID

Class pode ser reutilizada em vários elementos, enquanto o ID deve ser único em toda a página.

Prioridade no CSS (Cascata)

A ordem de prioridade é: Inline > ID > Classe > Elemento.

Principais Propriedades CSS

  • color e background-color: definem cores
  • font-size, font-family, font-weight: definem fontes
  • margin e padding: definem espaçamentos
  • border: define bordas
  • width e height: definem tamanhos
